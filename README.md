Social Media System

This project is a simple console-based social media app built in C++. It lets users register, log in, create posts, like and comment on posts, add friends, and send messages. All the data is saved in files so it stays even after closing the program. It also shows the total number of users on the welcome screen. Overall, it’s a basic but complete system that demonstrates how real social media platforms work using C++ concepts like OOP and file handling.
